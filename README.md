## 📌 Features
- ⚛️ Built with React (Vite setup)
- 🎨 Tailwind CSS for utility-first styling
- 📱 Fully responsive design
- 🧩 Modular components (`Card`, `Button`)
- 📑 Resume download button (optional)
- 📧 Contact section with GitHub and email
